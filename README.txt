User Manual & Instructions

1) Launch the file using a Java-supported IDE like Eclipse.
2) Execute the main project (APPGUI.java).
3) It is advisable to maximize the output window.
4) Select 'Select Root Folder & Copy' and choose the file containing the dataset (dataset folder).
5) Click on 'Index Images' to incorporate the images into the project database folder.
6) Click on the 'Minkowski's Distance' radio button to select the similarity measurement.
7) Click on 'Nearest 10' to retrieve the top-10 results.
